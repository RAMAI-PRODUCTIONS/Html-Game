# kenney_mini-skate

## Contents
- **GLB Models**: 20 files
- **Textures**: 1 files

## File Structure
```
kenney_mini-skate/
├── *.glb          # 3D model files in GLB format
└── Textures/      # Texture files referenced by the models
    └── *.png
```

## Usage
The GLB files are ready to use in any 3D application that supports the GLB format.
Textures are located in the Textures subfolder and are automatically referenced by the GLB files.
