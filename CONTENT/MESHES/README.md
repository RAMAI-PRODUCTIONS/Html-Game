# Kenney Mesh Assets

This folder contains 3D mesh assets from Kenney (www.kenney.nl).

## Organization

Each subfolder contains:
- **GLB files**: 3D models in GLB format (ready to use)
- **Textures folder**: PNG texture files referenced by the models
- **README.md**: Detailed information about the contents

## Usage

1. Navigate to the desired asset pack folder
2. Import the GLB files into your 3D application
3. Textures are automatically referenced by the GLB files

## Format

All models are in GLB format, which is:
- A binary version of glTF (GL Transmission Format)
- Widely supported by 3D applications and game engines
- Self-contained with embedded textures (when applicable)
